Descrição : Módulo de Haskell referente à terceira tarefa do projeto de LI. Copyright : Miguel Quaresma miguelquaresma97@gmail.com João Nogueira

Módulo que recebe um ficheiro contendo um mapa, coordenadas e uma linha com um caracter referente à direção pretendida do movimento do boneco, devolvendo, caso este movimento seja válido, as novas coordenadas do boneco ou, caso contrário, as coordenadas iniciais do boneco.
